# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any of the drivers or resources in this repository, please report it immediately.

**Contact us:**

- Email: [info@motioncomputing.com](mailto:info@motioncomputing.com)
- Phone: (+1) 800-355-1131

## Security Guidelines

- Do not publicly disclose vulnerabilities before contacting Motion Computing.
- Provide as much detail as possible: affected device model, driver version, steps to reproduce.
- For critical issues, use encrypted communication if possible.

We will respond promptly to all security reports and work to resolve issues as quickly as possible.
