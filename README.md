# Motion Computing – Intel Driver Repository
Welcome to the official Motion Computing Intel Driver repository. This repository provides official Intel driver packages for Motion Computing devices to ensure optimal performance, stability, and long-term support.

## Official Intel System Manufacturer

Motion Computing is listed as an **official System Manufacturer** on Intel's website.  
You can verify our status at [https://www.intel.com/content/www/us/en/support/system-manufacturers.html](https://www.intel.com/content/www/us/en/support/system-manufacturers.html).

## Purpose

This repository is designed to reduce the load on our main website by hosting verified Intel drivers for supported Motion Computing devices. 

## Supported Devices

- Motion X1 Series
- Motion X2 Series
- Legacy Models (see `/models/legacy`)

## How to Use

1. Navigate to the folder corresponding to your device model.
2. Download the required driver package(s) from the latest GitHub Release.
3. Verify the checksum in `/checksums/sha256.txt` to ensure integrity.
4. Follow the device installation instructions provided in each model's README.

> ⚠️ All drivers are unmodified official Intel releases. Motion Computing does not alter binaries.
