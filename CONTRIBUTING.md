# Contributing to Motion Computing Intel Driver Repository

We welcome contributions to improve documentation, add new device support, or enhance the repository structure. Please follow these guidelines:

## Reporting Issues

- Check existing issues before creating a new one.
- Provide clear device model and driver version.
- Include steps to reproduce problems if applicable.

## Submitting Updates

- All drivers must be official Intel releases.
- Include SHA256 checksum for any new driver added.
